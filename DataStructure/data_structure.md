Data structures are brad and butter of the Python language, used to organize and sort data

There are a few Built-in data structures in Python, for which no import is necessary.

Built-in types:

List → Ordered, Mutable collection of items.

Tuple → Ordered, Immutable collection of items.

Dictionaries → Unordered collection of key value pairs

Sets → Unordered collection of Unique items


The Collection module:

Double-Ended Queue → A list-like container with fast appends and pops on both ends.

Default-Dictionary → A dictionary that doesn't throw a KeyError if you try to access a key
                        that doesn't exist. Instead, it creates the key and assigns it a default value.

Counter → A dictionary subclass specifically designed for counting hashable objects.
